### ESP32 实时触摸传感器 Web 仪表盘实验 README
## 项目简介
本项目基于 **ESP32 + Web服务器 + 触摸传感器**，实现**实时物联网数据监测仪表盘**。通过网页 AJAX 异步刷新技术，实现触摸模拟量**无刷新、高频率、实时回显**，并配合 LED 进行状态指示，提供**三个递进版本代码**，从基础实时显示 → LED 联动 → 网页动态报警，完整覆盖物联网数据采集与展示核心知识点。

## 版本2：触摸阈值双灯联动（v2）
### 功能说明
1. **保留实时数据监测**
2. **增加阈值判断逻辑**（THRESHOLD = 400）
3. **双灯状态指示**：
   - 未触摸 → LED2 常亮
   - 触摸触发 → LED1 常亮
4. 硬件与网页**同步响应**

### 核心逻辑
```cpp
int tv = touchRead(TOUCH_PIN);
if (tv < THRESHOLD) {
  // 触摸：LED1 亮
  digitalWrite(LED1, HIGH);
  digitalWrite(LED2, LOW);
} else {
  // 未触摸：LED2 亮
  digitalWrite(LED1, LOW);
  digitalWrite(LED2, HIGH);
}
```

## 通用使用说明
### 1. WiFi 配置
三个版本统一使用相同 WiFi 信息：
```cpp
const char* ssid = "xswl";
const char* password = "hxy060617";
```

### 2. 上传与运行
1. 打开 `.ino` 代码文件
2. 选择 ESP32 开发板
3. 上传代码
4. 打开串口监视器获取 IP 地址
5. 浏览器输入 IP 进入实时仪表盘

### 3. 预期现象
1. 页面打开后**数字实时刷新**
2. 手指触摸 GPIO4 → **数值快速下降**
3. 手指离开 → **数值迅速回升**
4. 版本2/3：LED 随触摸状态同步变化
