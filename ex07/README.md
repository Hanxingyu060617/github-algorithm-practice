### ESP32 Web 控制实验合集 README
## 项目简介
本项目基于 **ESP32 开发板** + **Arduino Core 3.0+** 实现三个进阶 Web 控制实验，通过 WiFi（STA 模式 / AP 热点模式）搭建网页服务器，实现**远程 PWM 调光、双灯光效跷跷板、自动摇摆+手动控制一体化控制台**功能，涵盖前端交互、后端通信、PWM 驱动、状态同步等核心知识点，适合嵌入式 Web 控制入门学习。

---

## 实验一：单灯网页无极调光（ex07）
### 功能说明
1. ESP32 连接本地 WiFi，建立网页服务器
2. 网页端提供**滑动条**，实时控制板载 LED 亮度
3. 前端使用 `fetch` 异步发送数据，无刷新页面
4. ESP32 解析参数并通过 PWM 输出亮度

### 硬件接线
- LED 正极 → GPIO2
- LED 负极 → GND

### 使用说明
1. 修改代码中 WiFi 名称和密码
2. 上传代码，串口监视器查看 IP 地址
3. 浏览器输入 IP 打开控制面板，拖动滑块调节亮度

### 核心技术
- ESP32 Web 服务器路由配置
- HTML5 滑动条 + JavaScript 交互
- PWM 8 位分辨率驱动（0~255）
- 异步 HTTP 请求通信

## 实验二：双灯光效跷跷板（ex08）
### 功能说明
1. 双 LED 实现**亮度互补**（左亮右暗、左暗右亮）
2. 网页滑动条控制亮度重心
3. 前端实时显示双灯亮度数值
4. 单滑块控制双路 PWM 输出

### 硬件接线
- 左灯 LED → GPIO2
- 右灯 LED → GPIO4
- 共 GND

### 使用说明
1. 配置本地 WiFi
2. 上传代码，访问 IP 进入控制台
3. 拖动滑块观察双灯亮度反向变化

### 核心逻辑
```cpp
leftVal = 滑块值;
rightVal = 255 - 滑块值;
```

## 实验三：AP 热点 + 自动摇摆控制台（ex09）
### 功能说明
1. ESP32 开启独立 **WiFi 热点**，无需路由器
2. 支持**手动滑动控制** + **自动摇摆模式**一键切换
3. 自动模式下双灯自动往复渐变
4. 网页**实时同步硬件状态**，滑块与数值自动回显
5. 自动模式禁用滑块，防止逻辑冲突

### 硬件接线
- 左灯 → GPIO2
- 右灯 → GPIO4

### 使用说明
1. 手机/电脑连接 WiFi：`ESP32-LAB`
2. 密码：`12345678`
3. 浏览器访问：`http://192.168.4.1`
4. 点击按钮切换自动/手动模式

### 核心技术
- ESP32 AP 热点模式
- AJAX 轮询 + 硬件数据回传
- 全局状态机管理（自动/手动模式）
- 非阻塞自动渐变引擎

## 通用环境配置
### 开发环境
- Arduino IDE 2.0+
- ESP32 Arduino Core 3.0 及以上

### 依赖库
- `WiFi.h`（内置）
- `WebServer.h`（内置）

### PWM 驱动说明
本项目使用 **ESP32 新版 PWM API**：
```cpp
ledcAttach(pin, freq, resolution);  // 初始化
ledcWrite(pin, value);              // 输出占空比
```
兼容最新版 ESP32 内核，无需手动配置通道。

## 注意事项
1. 确保 ESP32 供电稳定，避免 LED 闪烁异常
2. AP 模式下仅支持连接 ESP32 热点的设备访问
3. 自动模式与手动模式互斥，防止控制冲突
4. 代码已适配 ESP32 Core 3.0+，低版本可能报错
