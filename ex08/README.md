### ESP32 物联网安防报警器实验 README
## 项目简介
本项目基于 **ESP32 + Web服务器 + 触摸传感器**，实现**物联网安防报警系统**，包含**布防/撤防、触摸触发报警、LED状态指示**三大核心功能。提供**三个递进版本代码**，从基础单灯报警 → 双灯交替闪烁 → 完整状态指示系统，逐步实现功能升级，完美匹配课程实验要求。

## 硬件接线
| 元器件         | ESP32 引脚 |
|----------------|------------|
| 触摸传感器引脚 | GPIO4      |
| 状态LED 1      | GPIO2      |
| 状态LED 2      | GPIO15     |
| 共用地线       | GND        |

## 版本1：双灯基础安防报警（v1）
### 功能说明
1. **网页布防/撤防**：通过网页按钮控制系统状态
2. **触摸触发报警**：布防状态下触摸引脚，锁定报警
3. **双灯独立指示**：
   - 状态灯（GPIO2）：布防时常亮
   - 报警灯（GPIO15）：报警时高速闪烁
4. **网页自动刷新**：实时显示系统状态

### 核心特性
- 基础状态机：`未布防 → 布防 → 报警 → 撤防`
- 非阻塞闪烁，不影响服务器运行
- 中文网页界面，状态颜色区分

---

## 版本2：双灯交替闪烁报警（v2）
### 功能说明
1. **极简代码结构**，适合入门学习
2. **双灯交替闪烁报警**：LED1亮时LED2灭，反向交替，视觉效果更强
3. 布防待命状态：双灯熄灭，低功耗等待
4. 报警状态锁定：触发后必须撤防才能停止

### 核心逻辑
```cpp
// 交替闪烁核心代码
digitalWrite(LED1, state);
digitalWrite(LED2, !state);
```

## 版本3：最终版安防系统（v3）
### 功能说明
**本项目完整版，推荐提交作业使用**
1. **三级状态指示**：
   - 已撤防：双灯全灭
   - 已布防（待命）：双灯慢速同步闪烁
   - 报警中：双灯高速交替闪烁
2. **美化响应式网页**：按钮样式、状态文字、自动刷新
3. 触摸灵敏度可调节（`threshold`阈值）
4. 系统状态稳定，无卡顿、无冲突

### 实验效果
- 撤防：无任何反应
- 布防：双灯慢闪，系统待命
- 触摸触发：双灯高速交替狂闪，锁定报警
- 撤防：停止报警，系统重置

## 通用使用说明
### 1. 配置WiFi
所有代码统一使用：
```cpp
const char* ssid = "xswl";
const char* password = "hxy060617";
```

### 2. 上传与运行
1. 打开 `.ino` 代码
2. 选择ESP32开发板
3. 上传代码
4. 打开串口监视器，获取IP地址
5. 浏览器访问IP，进入控制面板

### 3. 操作流程
1. 点击 **开始布防** → 系统进入警戒状态
2. 触摸GPIO4引脚 → 触发报警
3. 点击 **解除布防** → 停止报警，系统复位

## 核心技术知识点
1. **ESP32 Web服务器**：路由处理、网页刷新
2. **触摸传感器**：`touchRead()` 检测触摸信号
3. **状态机编程**：布防/报警/撤防状态管理
4. **非阻塞闪烁**：`millis()` 计时，不阻塞主程序
5. **HTML网页控制**：按钮、状态显示、页面跳转
